Each page on this section maps a runnable project in [`example/`](https://github.com/SaadArdati/plugin_kit/tree/main/example) to the concepts it demonstrates. The example directory is the source of truth; the page is the map. Read the page, run the code, swap things around. The three Flutter examples also ship as live web builds you can open without installing anything.
